# WebGPU Impostors using Threejs in TSL + R3F

### Implements Dynamic Octahedral impostors in threejs. Just provide a glb file and it will create an atlas and render it as an impostor.

It is under development. We will need to optimize performance to render hundreds of trees.

## Getting Started

Download and install Node.js on your computer (https://nodejs.org/en/download/).

Then, open VSCODE, drag the project folder to it. Open VSCODE terminal and install dependencies (you need to do this only in the first time)

```
npm install
```

Run this command in your terminal to open a local server at localhost:8080

```
npm run dev
```
